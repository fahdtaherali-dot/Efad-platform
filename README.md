#  Efad Platform (إيفاد)

Hi! I'm Fahd, a high school student, and this is my project: Efad.

**Why am I building this?**
I noticed that many smart students in the Arab world (especially those facing tough circumstances) have no idea how to apply to top global programs like Rise, Harvard, or MIT. They have the brains, but they lack the right guidance. 

Efad is a tool I'm developing to fix this. It basically takes a student's profile (GPA, skills, activities) and acts as a personal AI advisor. It points out what's missing in your application and gives you a clear plan to improve it.

**Current Status:**
This repo currently holds the froend UI (HTML/CSS/JS).
